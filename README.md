# RPN Calculator
A calculator written in java that uses a reverse Polish notation approach to solve given equations.

## Running the program
1. Open cmd/terminal depending on your operating system
2. Type ```java Main.java```

## Formatting
Commands are given in RPN (reverse Polish notation) eg. ```operand operand operator``` you can also stack calulations such as: ```1 1 + 2 *```. The calculator first performs addition: ```1 + 1 = 2``` then multiplication: ```2 * 2 = 4```.
